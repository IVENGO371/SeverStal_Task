Задание выполнялось для данных со следующей структурой:
 1. Уникальный идентификатор
 2. Символьная строка
 3. Числовое значение

Данные внутри каждой строчки должны быть указаны через запятую

Программа проверяет наличие следующих ошибок:
 1. Наличие идентификатора
 2. Формат идентификатора: он должен быть целым числом
 3. Повторение идентификаторов
 4. Наличие символьной строки
 5. Наличие числового поля
 6. Правильный формат числового поля: оно должно содержать только числа

Программа выводит отчёт в файл report.txt в следующем виде:
====================================
 Общее количество записей: 6
 Корректных записей: 3
 Некорректных записей: 3
 Уникальных идентификаторов: 3
 Общее количество ошибок: 4

 Агрегированная статистика (по корректным записям):
  Диапазон идентификаторов: 1 - 5
  Среднее значение value: 11.27
  Максимальное значение value: 16.00
  Минимальное значение value: 7.30

 Полный список ошибок (по типам):
  - ID: Дублирующийся идентификатор: 3
  - Name: Пустое название: 1

 Детализация некорректных записей:
  Строка 4: 5,Orange,17
      Ошибка: ID: Дублирующийся идентификатор

  Строка 5: 5,Poco,15
      Ошибка: ID: Дублирующийся идентификатор

  Строка 6: 5,,12
      Обнаружено 2 ошибок:
        1. ID: Дублирующийся идентификатор
        2. Name: Пустое название
